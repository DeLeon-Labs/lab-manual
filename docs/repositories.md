## Repository boundary

Plugin repositories own implementation choices, releases, tests, operational
documentation, and compatibility declarations. This manual owns ecosystem-wide
meaning, responsibility boundaries, shared contracts, and accepted decisions.

When a repository is created, renamed, transferred, or archived, update this map
in the same change that updates any affected charter or roadmap language.
